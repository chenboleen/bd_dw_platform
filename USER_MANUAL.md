# 用户操作手册

## 系统概述

bd_dw_platform 是一个企业级数据仓库元数据管理平台，帮助数据团队统一管理数据仓库中的表结构、字段信息、数据血缘关系和数据质量指标。

## 用户角色说明

| 角色 | 权限说明 |
|------|---------|
| ADMIN（管理员） | 拥有所有操作权限，包括创建、编辑、删除和管理用户 |
| DEVELOPER（开发者） | 可以查询所有数据，可以创建和编辑元数据注释 |
| GUEST（访客） | 只能查询和浏览，不能修改任何数据 |

## 登录系统

1. 打开浏览器，访问系统地址（默认：`http://localhost`）
2. 在登录页面输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后自动跳转到主页

> 默认管理员账号：`admin` / `admin123`


## 表元数据管理

### 查看表列表

1. 点击左侧菜单"表管理"
2. 页面展示所有表的列表，包含：表名、数据库、类型、描述、更新时间
3. 支持按数据库名、表类型过滤
4. 支持按表名搜索
5. 点击列头可排序

### 查看表详情

1. 在表列表中点击表名
2. 详情页展示：
   - 基本信息：表名、数据库、类型、描述、存储格式、存储路径
   - 字段列表：字段名、数据类型、是否可空、是否分区键、描述
   - 存储信息：数据大小、记录数

### 创建表元数据（DEVELOPER/ADMIN）

1. 点击"新建表"按钮
2. 填写必填信息：
   - 数据库名称
   - 表名称（同一数据库内唯一）
   - 表类型（TABLE/VIEW/EXTERNAL）
3. 填写可选信息：描述、存储格式、数据域（从目录下拉选择）
4. 存储路径由系统根据数据库名和表名自动生成，无需手动填写
5. 点击"确认"保存

### 编辑表元数据（DEVELOPER/ADMIN）

1. 在表详情页点击"编辑"按钮
2. 修改描述、存储格式、数据域等字段
3. 存储路径自动同步，无需手动修改
4. 点击"保存"

### 管理字段（DEVELOPER/ADMIN）

1. 在表详情页的字段列表中操作
2. 点击"添加字段"新增字段
3. 点击字段行的编辑图标修改字段信息
4. 拖拽字段行可调整字段顺序

## 血缘关系管理

### 查看血缘图谱

1. 点击左侧菜单"血缘关系"
2. 在搜索框输入表名，选择目标表
3. 选择查看方向：上游（数据来源）/ 下游（数据去向）/ 双向
4. 调整深度（1-5层）
5. 图谱以 DAG 可视化方式展示节点和连线，节点颜色含义：
   - 橙色：当前查询的中心表
   - 绿色：上游表（数据来源）
   - 蓝色：下游表（数据去向）
6. 节点标签显示 `数据库名.表名` 全名
7. 点击节点可查看表详情
8. 支持缩放和拖拽

### 创建血缘关系（DEVELOPER/ADMIN）

1. 在血缘关系页面点击"新建血缘"
2. 选择源表（数据来源）
3. 选择目标表（数据去向）
4. 选择血缘类型：DIRECT（直接）/ INDIRECT（间接）
5. 填写转换逻辑（可选）
6. 点击"确认"

### 影响分析

1. 在表详情页点击"影响分析"
2. 系统展示修改该表后会影响的所有下游表
3. 按层级展示影响范围

### SQL 解析提取血缘（DEVELOPER/ADMIN）

1. 在血缘关系页面点击"SQL 解析"
2. 粘贴 SQL 语句（支持 SELECT、INSERT、CREATE TABLE AS）
3. 点击"解析"，系统自动识别源表和目标表
4. 确认后保存血缘关系

## 全文搜索

### 基础搜索

1. 点击顶部搜索栏或左侧菜单"搜索"
2. 输入关键词（支持表名、字段名、描述）
3. 按回车或点击搜索图标
4. 结果高亮显示匹配内容

### 高级过滤

1. 在搜索页面展开"高级过滤"面板
2. 可按以下条件过滤：
   - 数据库名称
   - 表类型
   - 更新时间范围
   - 所有者
3. 多条件组合过滤

### 搜索建议

在搜索框输入时，系统自动显示匹配的表名建议，点击建议项快速跳转。

## 数据目录

### 浏览目录树

1. 点击左侧菜单"数据目录"
2. 左侧展示树形目录结构（最多5级）
3. 点击目录节点展开/折叠
4. 右侧展示选中目录下的表列表

### 管理目录（ADMIN）

1. 右键点击目录节点，选择操作：
   - 新建子目录
   - 重命名
   - 删除（需先移除所有表）
2. 拖拽目录节点可移动位置

### 将表添加到目录（DEVELOPER/ADMIN）

1. 在表详情页点击"添加到目录"
2. 在弹出的目录树中选择目标目录
3. 点击"确认"

## 数据质量

### 查看质量指标

1. 点击左侧菜单"数据质量"或在表详情页点击"质量"标签
2. 展示最新质量指标：
   - 记录数
   - 空值率（百分比）
   - 更新频率
   - 数据新鲜度（小时）
   - 综合质量分数

### 查看质量趋势

1. 在数据质量页面选择时间范围
2. 折线图展示各指标的历史趋势
3. 可对比多个时间段的质量变化

### 记录质量指标（DEVELOPER/ADMIN）

1. 在数据质量页面点击"记录指标"
2. 填写各项质量数据
3. 点击"保存"

## 变更历史

### 查看变更记录

1. 点击左侧菜单"变更历史"或在表详情页点击"历史"标签
2. 列表展示所有变更记录，包含以下字段：
   - 变更时间
   - 实体 ID
   - 实体名称（对应实体的名称）
   - 实体类型
   - 操作类型（创建/更新/删除）
   - 变更前后的值
   - 操作人（当前登录用户）
3. 支持按时间、实体类型等条件筛选

## 导入导出

### 导入元数据（DEVELOPER/ADMIN）

1. 点击左侧菜单"导入导出"，选择"导入"标签
2. 选择文件格式（CSV 或 JSON）
3. 点击"选择文件"上传（最大 100MB）
4. 系统验证文件格式并显示预览
5. 确认后点击"开始导入"
6. 查看导入结果（成功数、失败数、错误详情）

**CSV 格式要求：**
```
database_name,table_name,table_type,description
ods,user_info,TABLE,用户基础信息表
dwd,fact_order,TABLE,订单事实表
```

### 导出元数据

1. 点击"导出"标签
2. 选择导出格式（CSV 或 JSON）
3. 设置过滤条件（可选）
4. 点击"创建导出任务"
5. 在任务列表中查看导出进度
6. 完成后点击"下载"

## 常见问题

**Q：登录后页面空白？**
A：清除浏览器缓存后重试，或检查后端服务是否正常运行。

**Q：搜索没有结果？**
A：确认 Elasticsearch 服务正常运行，并且已有数据被索引。新创建的表需要等待几秒钟才能被搜索到。

**Q：血缘图谱加载很慢？**
A：减小查询深度（建议不超过3层），或缩小查询范围。

**Q：导入失败？**
A：检查文件格式是否符合要求，确认必填字段（database_name, table_name）不为空，表名在同一数据库内不重复。

**Q：没有权限执行某操作？**
A：联系管理员提升账号权限，或确认当前登录账号的角色。

## 最佳实践

1. **命名规范**：表名使用小写字母和下划线，如 `ods_user_info`
2. **描述填写**：为每张表和重要字段填写清晰的中文描述，方便团队协作
3. **血缘维护**：数据处理任务上线时同步维护血缘关系，保持血缘图谱准确
4. **定期质量检查**：建议每天记录一次质量指标，及时发现数据异常
5. **目录分类**：按数据分层（ODS/DWD/DWS/ADS）或业务域组织目录结构
