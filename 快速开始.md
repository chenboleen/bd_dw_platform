# 数据仓库元数据管理系统 - 快速开始指南

## 项目概述

这是一个全新的数据仓库元数据管理系统实现,位于 `kiro_bd_dw` 目录。

**技术栈:**
- 后端: Java 17+, Spring Boot 3.2+, MyBatis-Plus 3.5+, Spring Security 6.x
- 前端: Vue.js 3, TypeScript, Element Plus, Vite
- 数据库: MySQL 8.0+, Redis 7.x, Elasticsearch 8.x

## 环境准备

### 1. 安装 JDK 17+

```bash
# Windows (使用 Chocolatey)
choco install openjdk17

# 验证安装
java -version
```

### 2. 安装 Maven 3.8+

```bash
# Windows (使用 Chocolatey)
choco install maven

# 验证安装
mvn -version
```

### 3. 安装 MySQL 8.0+

```bash
# Windows (使用 Chocolatey)
choco install mysql

# 创建数据库
mysql -u root -p
CREATE DATABASE metadata_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 4. 安装 Redis 7.x

```bash
# Windows (使用 Chocolatey)
choco install redis-64

# 启动 Redis
redis-server
```

### 5. 安装 Elasticsearch 8.x

```bash
# Windows (使用 Chocolatey)
choco install elasticsearch

# 启动 Elasticsearch
elasticsearch
```

## 编译和运行

### 编译项目

```bash
cd kiro_bd_dw/backend-java
mvn clean install
```

### 运行应用

```bash
mvn spring-boot:run
```

### 访问应用

- Swagger API 文档: http://localhost:8080/swagger-ui.html
- 健康检查: http://localhost:8080/actuator/health
